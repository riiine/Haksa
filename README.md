## 학사관리프로그램 개인프로젝트
***
    학생정보를 관리하고 대출된 도서를 학과별로 확인하는 프로그램입니다.
    oracle DB에 학생, 도서대출에 대한 정보를 담아 저장하고
    이를 ojdbc를 통해 eclipse와 연동하였습니다.
    데이터의 입력, 수정, 삭제 기능을 구현했습니다.
    또한 jfreechart.jar 를 이용하여 학과별 대출횟수를 파이차트로 나타냈습니다.
#### BackEnd
    Java
#### DB
    Oracle db
